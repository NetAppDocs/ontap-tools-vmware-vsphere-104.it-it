= Aggiornare le credenziali di vCenter Server e ONTAP
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
È possibile aggiornare l'istanza di vCenter Server e le credenziali ONTAP tramite la console di manutenzione.

.Prima di iniziare
È necessario disporre delle credenziali di accesso dell'utente addetto alla manutenzione.

*Informazioni su questo compito*

Se hai modificato le credenziali per vCenter Server, ONTAP o Data LIF dopo la distribuzione, devi aggiornarle utilizzando questa procedura.

.Passi
. Dal vCenter Server, aprire una console per VASA Provider.
. Accedi come utente addetto alla manutenzione.
. Entra `2` per selezionare il menu Configurazione di sistema.
. Entra `9` per modificare le credenziali ONTAP .
. Entra `10` per modificare le credenziali di vCenter.
