= Reimposta la password utente dell'applicazione
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
La password utente dell'applicazione viene utilizzata per la registrazione di SRA e VASA Provider con vCenter Server.

.Passi
. Avviare ONTAP Tools Manager da un browser web: `\https://<ONTAPtoolsIP>:8443/virtualization/ui/`
. Accedi con le credenziali di amministratore ONTAP tools for VMware vSphere fornite durante la distribuzione.
. Seleziona *Impostazioni* dalla barra laterale.
. Nella schermata *Credenziali VASA/SRA*, seleziona *Reimposta password*.
. Inserisci una nuova password e conferma i nuovi dati inseriti.
. Selezionare *Reimposta* per implementare le modifiche.
